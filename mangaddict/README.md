
Installation

1. **Base de données** : exporter la bdd
   - Vérifiez que le fichier `config.ini` contient vos identifiants.

2. **Lancement**:
   - placez-vous dans le dossier du projet via le terminal.
   - Lancez : `php -S 127.0.0.1:3004 -t public`
   - Ouvrez le navigateur sur `http://127.0.0.1:3004`.
